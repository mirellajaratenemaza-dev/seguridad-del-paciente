
# SafePatient IA (Frontend)

## Requisitos
- Node 18+

## Instalar
```bash
cd frontend
npm install
```

## Configurar API
Por defecto el frontend apunta a `http://localhost:8000`.
Si necesitas cambiarlo:
- crea un archivo `.env` dentro de `frontend/` con:
```
VITE_API_BASE=http://localhost:8000
```

## Ejecutar
```bash
npm run dev
```
